# 游戏机制详解

## 战斗系统

### 非线性敌人强度缩放

敌人属性随游戏天数非线性增长（替换旧线性公式 `1+(day-1)/500`）：

| 游戏天数 | 缩放阶段 | 效果 |
|----------|----------|------|
| 1-100 | 线性初期 | 缓慢增长（0~0.2） |
| 100-300 | 对数中期 | 快速提升（0.2~0.5） |
| 300-700 | 线性后期 | 稳步增长（0.5~0.8） |
| 700+ | 渐近天阶 | 趋向上限（0.8~1.0） |

### 天阶敌人多阶段血条

天阶敌人拥有 3 阶段血条：

| 血量阈值 | 阶段描述 | 效果 |
|----------|----------|------|
| 100%-50% | 普通 | 正常攻速 |
| 50%-20% | 浑身浴血 | 攻速 +30% |
| <20% | 濒死狂化 | 攻速 +50%，额外一击 |

### 回合流程

```
玩家行动 → 计算伤害/效果 → 敌人回合 → 检查胜负 → 循环
```

### 玩家动作

| 动作 | 说明 |
|------|------|
| `basic_attack` | 普通攻击，伤害 = 力量 × 0.8 |
| `defend` | 防御，本回合受伤害减半，回合末回复少量生命 |
| `martial` | 释放武学，消耗体力/内力，有冷却时间 |
| `useItem` | 使用背包中的消耗品 |
| `flee` | 逃跑，成功率与身法相关 |

### 武学伤害公式

```
伤害 = 基础伤害 × 武学damage倍率 × (1 + 力量加成) + 武器伤害
```

### 武学冷却

| 等阶 | 冷却回合 |
|------|----------|
| 天 | 3 |
| 地 | 4 |
| 玄 | 5 |
| 黄 | 6 |
| 无 | 8 |

### 战斗日志类型

- `combat`（红色加粗边框）— 攻击与伤害
- `event`（金色边框）— 战斗事件
- `enemy`（灰色斜体）— 敌人行为
- `system`（灰色）— 系统提示

### 天阶敌人警告

当遭遇天阶敌人时，屏幕闪现红色全屏遮罩（2 秒），显示"⚠ 天阶强敌来袭 ⚠"。

## 武学共鸣系统

装备两套心法时，若同属一元素，触发共鸣加成（见 data-modules.md 共鸣表）。

共鸣效果实时生效，影响所有战斗行为。

## 任务系统

### 告示栏操作

- **查看新任务** — 每次消耗 1 回合，生成 3 条任务（按等阶概率分布）
- **任务上限** — 同时最多 5 条进行中任务
- **领取奖励** — 任务完成后按钮出现，点击后发放奖励

### 任务过期

每 24 刻检查一次，`checkTaskExpiration()` 清理过期任务并应用惩罚：

| 等阶 | 过期惩罚 |
|------|----------|
| 天 | 属性永久受损（力量-3，气海-3） |
| 地 | 扣除 500 两银子 |
| 玄 | 扣除 24 小时寿命 |
| 黄 | 扣除 100 两银子 |
| 无 | 无额外惩罚 |

### 连环任务

完成特定任务后 20% 概率触发连环任务续接（4 阶段：起→承→转→合），最终击败 Boss。

### 任务显示（NoticeView）

每张任务卡片包含：
- 等阶标签与颜色边框
- 任务名称与描述
- 目标进度（当前/总数）
- 时限倒计时（颜色随紧迫度变化）
- 失败惩罚提示
- 蝴蝶效应说明（若有）
- 领取按钮（完成后显示）

## 世界事件

### 红潮日

- 触发概率：2%（每 24 刻检查）
- 持续时间：48 小时（192 刻）
- 效果：所有建筑背景泛红，黑市价格减半，遇敌率翻倍

### 物价波动

- 触发概率：累计 3%（每 24 刻检查）
- 随机选择区域，价格涨跌 20%
- 持续至下一次物价事件覆盖

### 建筑敌意

- 来源：完成地阶及以上任务产生的蝴蝶效应
- 效果：药铺/铁匠铺/酒馆拒绝服务
- 持续：72 小时（factionHostility 字段）
- 过期：与 worldEffects 同步清理

## 时间系统

```
1 刻 = 15 分钟
1 时辰 = 8 刻
1 天 = 96 刻
总时长 = 1000 天（96000 刻）
```

### 时间流逝

- 每次移动（探索/移动方向）：+2 刻
- 每次战斗：+1~5 刻（视战斗长度）
- 每次告示栏刷新：+1 刻
- 每次区域传送：+8 刻

### 自然恢复

- 体力：每刻 +0.5
- 内力：每刻 +0.25
- （卧薪尝胆/心法加成可提高回复速率）

| 耐力 | 体力上限、防御力 |

## 存档自动触发时机

`saveToStorage()` 在以下时机自动调用：

1. `startFate()` — 创建新角色时
2. `createCharacter()` — 命格问题完成后
3. `advanceTurn()` — 每回合推进时
4. `claimQuestReward()` — 领取任务奖励后
5. `learnMartial()` — 学会武学时
6. `learnProcMartial()` — 学会过程化武学时
7. `equipItem()` / `unequipItem()` — 装备变更时
8. `upgradeEquipment()` — 强化装备后
9. `travelToRegion()` — 跨区域传送时
